Getting started with LEX

Welcome to the LEX Tutorial. In this we will cover some basics of LEX.

Installing FLEX

For Ubuntu, install using:

        
        $ sudo apt-get install flex

Example LEX File

Let's start with a classic example: Seperating words and numbers.

        
    /* LEX specification for the Example */
    %%
    [0-9]+   { printf("NUMBER: %s\n", yytext); }
    [a-zA-Z]+   { printf("WORD: %s\n", yytext); }
    %%
    int yywrap(){}
    int main(){yylex();}

Compiling

Save your LEX file with `.l` . Run `flex` command to generate C source code of the lexer.

        
    $ flex lex.l

Now you need to compile the generated source code (`lex.yy.c`) to create an executable.

            
    $ gcc lex.yy.l -o lexer -lfl

After compilation, you can run the generated executable to tokenize input files:

                
    $ ./lexer input.txt
